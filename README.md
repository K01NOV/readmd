# Чек-лист проверки сайта колледжа

---

## Проверка формы Онлайн приёмной

- [ ] Проверить наличие формы на странице (ведёт ли она к правильной секции или отдельной странице)
- [ ] Проверить отображение всех полей (Фамилия, Имя, Отчество, Телефон, Электронная почта, Вопрос/Сообщение)
- [ ] Проверить наличие обязательных полей и их визуальное выделение
- [ ] Проверить работу поля "Телефон" — ввод корректных номеров
- [ ] Проверить работу поля "Электронная почта" — ввод корректных и некорректных адресов
- [ ] Проверить валидацию полей (например, пустое поле, неправильный формат Email, некорректный номер телефона)
- [ ] Проверить работу кнопки "Отправить" при заполненных корректных данных
- [ ] Проверить работу кнопки "Отправить" при неправильных/пустых данных — получение ошибок/подсказок
- [ ] Проверить отображение сообщения об успешной отправке формы
- [ ] Проверить работу защиты от спама (капча или аналогичные меры, если есть)
- [ ] Проверить очистку формы после успешной отправки
- [ ] Проверить, что при ошибке формы не происходит сброс введённых данных без уведомления
- [ ] Проверить, что отправка формы происходит через безопасное соединение (HTTPS)
- [ ] Проверить, что в случае сетевой ошибки показывается сообщение об ошибке
- [ ] Проверить, что отправленные данные попадают в базу или систему обработки (если есть возможность)
- [ ] Проверить работу формы при отключённом JavaScript (если предусмотрено)
- [ ] Проверить корректность работы формы на мобильных устройствах и разных браузерах

---

## Проверка поиска на сайте

- [ ] Проверить отображение поисковой строки на всех страницах
- [ ] Ввести существующие в базе слова и убедиться, что результаты отображаются
- [ ] Ввести несуществующие слова — должно отображаться сообщение "ничего не найдено"
- [ ] Провести поиск по частичным совпадениям
- [ ] Проверить работу поиска при использовании спецсимволов
- [ ] Проверить, что поисковая строка очищается после поиска
- [ ] Проверить работу функции автодополнения (если есть)
- [ ] Проверить результаты поиска на мобильных устройствах
- [ ] Проверить работу поиска при отключённом JavaScript (если есть асинхронные функции)
- [ ] Проверить, что поиск не вызывает ошибок браузера
- [ ] Проверить, что по клику на результат поиска происходит переход на правильную страницу
- [ ] Проверить работу поиска по разным разделам сайта (если есть фильтры или категории)
- [ ] Проверить критерии релевантности результатов поиска
- [ ] Проверить отображение пагинации результатов поиска
- [ ] Провести стресс-тест — ввод очень длинных поисковых запросов
- [ ] Проверить, что поиск отображается на всех устройствах и браузерах

## Проверка шапки сайта

- [ ] Проверить наличие логотипа и его кликабельность (переход на главную)
- [ ] Проверить корректность отображения меню навигации
- [ ] Проверить адаптацию шапки на мобильных устройствах
- [ ] Проверить наличие и работу кнопок входа/регистрации
- [ ] Проверить наличие контактов или иконок соцсетей
- [ ] Проверить отображение и функционирование иконки "Обратная связь" или "Задайте вопрос"
- [ ] Проверить наличие поиска в шапке (если есть)
- [ ] Проверить отображение элементов на разных браузерах
- [ ] Проверить, что шапка остается фиксированной или правильно прокручивается (при необходимости)
- [ ] Проверить отображение шапки при низких разрешениях экрана
- [ ] Проверить работу выпадающих меню (если есть)
- [ ] Проверить наличие элементов скорости загрузки и их работы при быстром просмотре
- [ ] Проверить, что логотип и меню не перекрываются и не исчезают при различных разрешениях

---

## Проверка главной страницы сайта

- [ ] Проверить наличие основного баннера/слайдера — его функционирование и изображений
- [ ] Проверить правильность отображения ключевых разделов ("О колледже", "Магистратура", "Абитуриентам" и др.)
- [ ] Проверить работоспособность ссылок и их корректное ведение на соответствующие разделы
- [ ] Проверить отображение новостей или анонсов на главной
- [ ] Проверить работу элементов "Преимущества", "Новости", "Партнёры" и других информационных блоков
- [ ] Проверить наличие и функционирование блока "Контакты" и карты
- [ ] Проверить адаптивность главной страницы (мобильные/планшеты)
- [ ] Проверить отображение элементов при отсутствии интернет-соединения или с медленной загрузкой
- [ ] Проверить загрузку изображений и мультимедийных элементов
- [ ] Проверить наличие и правильность работы формы подписки (если есть)
- [ ] Проверить корректность отображения и работу элементов поиска
- [ ] Проверить отображение футера (контакты, копирайт, ссылки соцсетей)
- [ ] Проверить работу всех CTA-кнопок (Записаться, Подробнее, Читать далее и др.)
- [ ] Проверить, что контент не перекрывается и не идет за границы
- [ ] Проверить скорость загрузки главной страницы
- [ ] Определить наличие возможных багов визуального отображения или ошибок JavaScript
- [ ] Проверить верстку искажений или сбоев на разных браузерах и устройствах

## Бонус — возможные баги и идеи для поиска недочётов

- [ ] Проверить наличие и правильную работу языковых переключателей (если есть)
- [ ] Ввести неправильные данные в формы и убедиться в корректных сообщениях об ошибках
- [ ] Попытаться отправить формы с отключённым интернетом — проверить поведение системы
- [ ] Проверить работу сайта в режиме инкогнито/без авторизации
- [ ] Проверить работу сайта при очень медленном соединении (simulate network throttling)
- [ ] Исследовать возможные уязвимости по безопасности (например, XSS, SQL Injection в форму)
- [ ] Проверить загрузку и корректность отображения сторонних скриптов и виджетов
- [ ] Вести аудит возможных утечек информации или приватных данных при ошибках
